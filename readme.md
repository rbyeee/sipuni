# СИПУНИ

### Установка зависимостей

```bash
npm i
```

### Запуск в режиме разработки

```bash
npm run dev
```

## Доступные команды

- `yarn dev` - запуск в режиме разработки
- `yarn build` - сборка для продакшена

```
app/
├── fonts/        # Шрифты
├── image/dist/   # Сюда можешь грузить картинки
├── js/app.js     # Index скрипты (папка db) есть остальные скрипты + скрипт под страница amocrm
├── parts/        # Пытался переносить используемые компоненты
├── styles/sass   # Стили под страницы
```
